# Security Policy

## Overview

AetherLens is an educational and portfolio project demonstrating modern network analysis techniques using Rust, .NET, and Next.js.

**Note:** This project is primarily for demonstration purposes and is not actively maintained for production use.

## Reporting a Vulnerability

Since this is a portfolio project:

1.  **Please use GitHub Issues:** You are welcome to open an issue for any security vulnerabilities or bugs you find. This helps document improvements for future reference.
2.  **No Embargo:** You do not need to wait for a fix before disclosing, as this software is not intended for critical production environments.
3.  **Educational Value:** We appreciate detailed reports that explain the vulnerability, as they provide valuable learning opportunities.

## Supported Versions

This project is provided "as-is" without warranty of any kind. Use at your own risk.

## Safe Harbor

We support safe harbor for security research. We will not pursue legal action against anyone conducting good-faith security research on this project.
